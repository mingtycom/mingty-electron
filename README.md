# mingty-electron

> 위 프로젝트는 svelte와 electron 템플릿 프로젝트입니다. by mingty

> - svelte 3.0.0
> - electron 22.0.0

### 모듈 설치
``` sh
$ npm i
```

### 프로젝트 빌드 전
``` sh
* 로컬
/사용자/.cache/puppeteer/chrome/

* 로컬 프로젝트
/프로젝트 ROOT/node_modules/puppeteer/chrome/**/*

*로컬 => 로컬 프로젝트 위치에 복사하여야함.
```

### 빌드
``` sh
$ npm run build
```

### 프로젝트 실행(svelte)
``` sh
$ npm run dev
```

### 프로젝트 실행(electron)
``` sh
$ npm run e
```
